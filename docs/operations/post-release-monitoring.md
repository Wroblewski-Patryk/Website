# Post-Release Monitoring

## First 10 Minutes
- health endpoints
- error rate
- latency
- queue depth
- alert thresholds and ownership routing from `docs/operations/production-alerting-thresholds-and-ownership.md`

## First 24 Hours
- incidents and alerts summary
- user-impacting defects
- rollback decision log (if any)

## Follow-up
- remediation tasks
- documentation updates
- retrospective notes
